在cmd里面

输入：
git config -l (查看全部配置)


输入：
git config --system --list (查看系统配置)

输入：
git config --global --list (查看自定义的全局配置)

关键配置->输入：(设置自定义的全局关键配置项 user.name和user.email)
git config --global user.name "ZWN95" 
git config --global user.email "2740770492@qq.com"


新建git项目，在文件夹路径栏输入cmd，进入cmd后
输入：
git init
或输入：(克隆远程仓库)
git clone [url]

git status 查看add状态

git add . 工作区/工作目录下的所有文件添加到暂存区，点号代表所有

git commit -m "这里写一些日志信息" 从暂存区保存到本地仓库

git push 本地仓库推送到云仓库

git branch 查看分支

git branch [name] 新建分支

git branch -d [name] 删除分支

git checkout [name] 切换分支

git merge [name] 合并到当前分支

git push -u origin [name] 新建本地和云仓库分支，并推送到云仓库

git fetch 从云仓库下载到本地仓库

git pull 从云仓库下载到到工作区/工作目录

git reset 从本地仓库下载到暂存区

git checkout 从暂存区下载到工作区/工作目录

工作区/工作目录-> 暂存区-> 本地仓库-> 云仓库
